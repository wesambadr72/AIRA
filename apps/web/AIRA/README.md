# AIRA Web Application 🚀

This directory contains the main web application frontend for AIRA (AI Research Assistant).

For complete information about the project features, installation, project structure, and usage, please refer to the main repository README:
👉 **[Root Repository README](../../../README.md)**

### Quick Commands:
- **Install dependencies**: `npm install`
- **Run dev server**: `npm run dev`
- **Build production bundle**: `npm run build`
